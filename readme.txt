# Agentic Image Processor 🩺

An intelligent pipeline that identifies medical use cases using DICOM metadata.

## 📁 Project Structure
- `config/`: Use case mappings.
- `data/raw/`: Original medical images (DICOM/NIfTI).
- `src/`: The Agentic logic and processing modules.
